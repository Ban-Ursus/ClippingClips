2021. 2. 4 ~ 6. sw중심 해커톤  ClippingClips.
https://youtu.be/zeaql9gDb0A

# Feature
1. 들어갈 때마다 영상을 랜덤으로 뽑아옴.
2. 한글로 검색하면 파파고번역을  통해 영어로 번역되고 이를 데이터베이스에서 찾아 뿌려줌.
3. 저장 버튼을 누르면 나만의 암기장으로 영상 저장.
4. STT를 사용해 스피킹테스트(말한 발음 체크용)
